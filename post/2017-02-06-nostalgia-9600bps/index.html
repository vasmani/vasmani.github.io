PC통신의 추억 - 곰주인 호스트 - 이야기5.3

옛날 옛적 20년도 넘은 그 오래 전! 그때의 PC통신의 추억으로 빠져 보다! 효린이용 PC만들어 주기 프로젝트중, dosbox에 재미있는 기능들이 있어서 한 번 해 보았다.

Server

  • Raspberry Pi Model B
  • Raspian
  • Dosbox
[modem]
modem=true
comport=1
listenport=23

[serial]
serial1=modem listenport:5000
serial2=dummy
serial3=disabled
serial4=disabled

  • OSX
  • Dosbox

serial1=modem listenport 23

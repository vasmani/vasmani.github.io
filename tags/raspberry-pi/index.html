    5천원 짜리 컴퓨터 라즈베리파이 제로

    라즈베리파이 제로를 테스트해 보았다. 영국 소재 Pimoroni 쇼핑몰에서 3.33파운드(4,800원 정도)에 구매했다. 배송비가 5.5파운드(8,000원 정도)이긴 하지만, 대략 1만 3천원 정도에 득템! 특수 모양의 USB커넥터가 필요한데, 예전에 우연히 알리에서 구매해 놓은 것이 있었다! USB포트가 없기 때문에 USB허브 연결이 필수다. 연결해 보니 잘 동작한다. CPU/RAM 사양이나 성능은 2013년에 구매했던 Raspberry Pi 1 Model B 보다 좋은 것 같다. 꽤 쓸만 함. GPIO가 많다(40핀). 근데 핀을 납땜해야 한다. 다양한 사용을 위해서는 유전원 USB허브, WIFI동글, 블루투스동글, HDMI어댑터 같은 것들이 필요하니… 스트레스 없이 다양한 이용은 아무래도 Raspberry Pi 3가 현재로서는 최고.
